## Pythonと外部ライブラリ

ソースコードを実行するには下記のライブラリが必要です。

* NumPy
* Matplotlib
* PyTorch（バージョン：2.x）
* torchvision
* tqdm

※Pythonのバージョンは **3系** を利用します。


## 実行方法

各章のフォルダへ移動して、Pythonコマンドを実行します。

```
$ cd step01
$ python norm_dist.py

$ cd ../step02
$ python generate.py
```
